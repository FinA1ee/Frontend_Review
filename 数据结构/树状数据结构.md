二叉树 Binary Tree:
  1. 特点: 每个node至多有两个children
  2. 性质: 完全二叉树 H = log2n + 1
  3. 遍历:
    - 前序遍历 PreOrder:  根 -> 左 -> 右
    - 中序遍历 InOrder:   左 -> 根 -> 右
    - 后序遍历 PostOrder: 左 -> 右 -> 根
  4. 效率
    - 动态查找/插入: O(h), worst case: O(n)

二叉搜索树 Binary Search Tree:
  1. 特点: 
    - 左子树上的所有值小于根
    - 右子树上的所有值大于根
  2. 操作:
    插入/查找/删除
  3. 效率

  适合处理批量动态数据，文件系统和数据库系统一般都采用树（特别是B树）的数据结构数据，主要为排序和检索的效率。
